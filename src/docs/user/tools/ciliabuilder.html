Tool: CiliaBuilder

The CiliaBuilder Tool is a ChimeraX package used to generate parameterized and custom 3D models of microtubule-based structures, primarily cilia and centrioles. It provides high-level commands for quick model creation and a low-level command for detailed geometry import.

The primary functions of the tool are exposed through three ChimeraX commands:

  • ciliabuild Command:

    Used to generate a complete cilia or flagellum structure (typically 9+2 axoneme). This command allows the user to specify the centerline (straight, curved, sinusoidal, or template), overall length, microtubule radii, central pair presence, tip termination style, and membrane parameters.

  • centriolebuild Command:

    Used to generate a complete centriole structure with microtubule triplets (typically 9x3 configuration). It automatically calculates the characteristic distal termination of the B- and C-tubules and allows customization of length, radius, and overall angular offset.

  • ciliabuild_from_csv Command:

    Provides maximum flexibility by loading detailed 3D microtubule geometry data from a custom CSV file. This allows users to import complex, pre-calculated coordinates and termination flags for all components (doublets, central pair, membrane, cap) of any microtubule-based structure.
