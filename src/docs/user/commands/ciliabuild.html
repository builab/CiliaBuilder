ciliabuild

Generate a complete cilia structure (typically 9+2 axoneme) as a ChimeraX model. The structure includes microtubule doublets, an optional central pair, an optional tip complex, and an optional membrane.

Usage

ciliabuild [Centerline_Options] [Structure_Options] [Geometry_Options] [Tip_Options] [Membrane_Options]

Examples

Note: All length and radius parameters use Ångströms (Å).

Simple straight cilia (9+2)

ciliabuild length 10000

Curved cilia

ciliabuild length 10000 type curve curve_radius 10000

Sinusoidal cilia

ciliabuild length 15000 type sine sine_frequency 1 sine_amplitude 1000

Cilia with defined tip length and random termination

ciliabuild length 10000 type tip tip_length 4000
For this option, the tip is generated randomly. Therefore, a csv file ciliatip.csv is generated containing information to replicate the exact model generated using ciliabuild_from_csv

Primary cilia (9+0)

ciliabuild length 10000 type primarycilia draw_central_pair false
For this option, the primary cilia is generated randomly. Therefore, a csv file primary.csv is generated containing information to replicate the exact model generated using ciliabuild_from_csv

Using 2D template file

ciliabuild length 10000 type template template_file template.csv

Custom number of doublets (e.g., 12+2)

ciliabuild length 5000 num_doublets 12 cilia_radius 1000

Options

Centerline Options

length float
Total length of the cilium/flagellum in Ångströms. (default: 10000.0)
type string
The type of centerline geometry: straight, curve (circular arc), sine (sinusoidal wave), primarycilia, or template (load from custom CSV). (default: straight)
curve_radius float
Radius of curvature (in Å) used for the curve line type. (default: 10000.0)
sine_frequency float
How many times of a sine cycle for the sinusoidal line type. (default: 1.0)
sine_amplitude float
Amplitude (in Å) for the sine line type. (default: 5000.0)
template_file string
Filename for the 2D centerline template CSV if line is template. (default: 'template.csv')

Axoneme Structure Options

num_doublets integer
Number of doublet microtubules. (default: 9)
cilia_radius float
Radius of the axoneme barrel (distance from the center axis to the doublet A-tubule centerline) in Å. (default: 875.0)
draw_central_pair boolean
If True, draws the two central singlet microtubules (the '+2'). (default: True)

Microtubule Geometry & Color Options

doublet_a_radius float
Radius of the A-tubule in Å. (default: 125.0)
doublet_b_radius float
Radius of the B-tubule in Å. (default: 135.0)
doublet_shift float
The centerline shift magnitude (in Å) defining the A/B overlap. (default: 70.0)
doublet_a_color color
RGBA color for the A-tubules. (default: #6464FF)
doublet_b_color color
RGBA color for the B-tubules. (default: #6464FF)
cp_radius float
Radius of the Central Pair (CP) singlet microtubules in Å. (default: 125.0)
cp_shift float
The centerline shift magnitude (in Å) separating the two central pair singlets. (default: 160.0)
cp_color color
RGBA color for the Central Pair microtubules. (default: #FFFF64)

Tip Options

tip_length float
Length of the distal tip region in Å where doublets may terminate. (default: 2000.0)

Membrane Options

membrane boolean
If True, draws a semi-transparent cylindrical membrane surface. (default: True)
membrane_radius float
Radius of the membrane cylinder in Å. (default: 1100.0)
membrane_fraction float
Fraction of the total length that the membrane covers. (default: 0.7)
membrane_color color
RGBA color for the membrane (default: #666666)
