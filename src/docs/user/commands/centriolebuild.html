centriolebuild

Generate a complete centriole structure with triplet microtubules (typically 9x3 configuration) as a ChimeraX model. Centrioles are rendered with specific length differences for the B- and C-tubules to model the distal end fade-out.

Usage

centriolebuild [Centerline_Options] [Structure_Options] [Triplet_Geometry_Options]

Examples

Note: All length and radius parameters use Ångströms (Å).

Basic straight centriole (9x3)

centriolebuild length 3000

Curved centriole

centriolebuild length 5000 type curve curve_radius 8000

Centriole with custom angle offset (e.g., centriole_angle_offset = 0 to match an aligned cilia)

centriolebuild length 5000 centriole_angle_offset 0

Centriole with custom termination lengths

centriolebuild length 4000 triplet_b_length_diff 500 triplet_c_length_diff 1000

Custom number of triplets and radius (e.g., 12x3)

centriolebuild length 5000 num_triplets 12 centriole_radius 1200

Options

Centerline Options

length float
Total length of the centriole in Ångströms. (default: 4000.0)
type string
The type of centerline geometry: straight, curve, or sine. (default: straight)
curve_radius float
Radius of curvature (in Å) used for the curve line type. (default: 10000.0)
sine_frequency float
How many times of a sine cycle for the sinusoidal line type. (default: 1.0)
sine_amplitude float
Amplitude (in Å) for the sine line type. (default: 2000.0)
template_file string
Filename for the 2D centerline template CSV if line is template. (default: 'template.csv')
z_offset_end float
The target Z-coordinate where the *end* (most distal point) of the centriole should be placed. The entire structure is shifted to meet this value. (default: 0.0)

Centriole Structure Options

num_triplets integer
Number of microtubule triplets (e.g., 9). (default: 9)
centriole_radius float
Radius of the centriole barrel (distance from center to A-tubule centerline) in Å. (default: 875.0)
centriole_angle_offset float
Angular offset (in degrees) for the entire 9-fold structure. (default: 30.0)

Triplet Geometry & Color Options

triplet_a_radius float
Radius of the A-tubule in Å. (default: 125.0)
triplet_b_radius float
Radius of the B-tubule in Å. (default: 135.0)
triplet_c_radius float
Radius of the C-tubule in Å. (default: 135.0)
triplet_ab_shift float
The centerline shift magnitude (in Å) from the center of A & B. (default: 70.0)
triplet_c_shift float
The centerline shift magnitude (in Å) applied to the C-tubule relative to the center. (default: 200.0)
triplet_b_length_diff float
The length (in Å) by which the B-tubule is shorter than the A-tubule (for distal termination). (default: 20.0)
triplet_c_length_diff float
The length (in Å) by which the C-tubule is shorter than the A-tubule (for distal termination). (default: 40.0)
triplet_a_color color
RGBA color for A-tubules. (default: ##6464FF)
triplet_b_color color
RGBA color for B-tubules. (default: ##6464FF)
triplet_c_color color
RGBA color for C-tubules. (default: #B3B3FF)
